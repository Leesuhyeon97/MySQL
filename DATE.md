<H1>DATE</H1>

> MySQL에서 날짜 사용법



1.DATE_FORMAT

> DATE를 다른 형식으로 변형할 때 사용

| 구분기호 | 역할                | 구분기호 | 역할                    |
| -------- | ------------------- | -------- | ----------------------- |
| %Y       | 4자리 년도          | %m       | 숫자 월 (무조건 두자리) |
| %y       | 2자리 년도          | %c       | 숫자 월 (한자리,두자리) |
| %M       | 긴 월 (영문표기)    | %d       | 일자 (무조건 두자리)    |
| %b       | 짧은 월(영문표기)   | %e       | 일자 (한자리, 두자리)   |
| %W       | 긴 요일 (영문표기)  | %I       | 시간 (12시간)           |
| %a       | 짧은 요일(영문표기) | %H       | 시간 (24시간)           |
| %i       | 분                  | %r       | hh : mm : ss AM,PM      |
| %T       | hh : mm : ss        | %S       | 초                      |
